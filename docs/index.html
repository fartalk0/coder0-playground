Qwen Coder Auto-Merge

Этот репозиторий настроен на автоматическое принятие Pull Request'ов от Qwen Coder.

Все изменения, предложенные Qwen Coder, будут автоматически подтверждаться и вливаться в основную ветку благодаря GitHub Actions workflow.

Используется стратегия squash merge с автоматическим удалением ветки после слияния.

Статус: Автоматический мердж активен
